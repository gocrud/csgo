# 阶段 1：核心基础

[← 返回主目录](../../README.md)

本阶段将深入讲解 CSGO 框架的核心基础知识，为构建生产级应用打下坚实基础。

## 学习目标

完成本阶段后，你将能够：

- ✅ 深入理解依赖注入系统
- ✅ 掌握路由和处理器的高级用法
- ✅ 熟练使用配置系统
- ✅ 构建完整的 RESTful API
- ✅ 完成一个简单的实践项目

## 学习时长

⏱️ 预计时间：4-6 小时

## 学习路径

### 1. [依赖注入详解](dependency-injection.md)
- DI 容器工作原理
- Singleton 生命周期
- 服务注册和解析
- 自动依赖解析
- 最佳实践

### 2. [Web 应用基础](web-basics.md)
- WebApplicationBuilder 详解
- 应用生命周期
- 环境管理
- 主机配置

### 3. [路由系统](routing.md)
- 路由定义和匹配
- 路径参数和查询参数
- 路由组和嵌套
- 路由约束
- 最佳实践

### 4. [配置管理](configuration.md)
- 配置源（JSON、环境变量、命令行）
- 配置绑定
- Options 模式
- 配置热更新
- 最佳实践

### 5. [HTTP 上下文](http-context.md)
- HttpContext 详解
- 请求和响应处理
- ActionResult 模式
- 请求绑定
- 响应格式化

### 6. [实践项目：简单 API](project-simple-api.md)
- 需求分析
- 项目结构
- 实现用户管理 API
- 测试和调试

## 前置要求

- 完成[阶段 0：快速入门](../00-getting-started/)
- 熟悉 Go 语言基础
- 了解 HTTP 协议

## 学习建议

1. **循序渐进**：按顺序学习每个主题
2. **动手实践**：运行所有示例代码
3. **深入理解**：理解概念而不只是记忆
4. **完成项目**：认真完成实践项目
